# TileGallery - Find Your Perfect Aesthetic

Welcome to **TileGallery**, a premium web application designed for browsing and exploring exquisite tile designs for modern homes. Whether you're looking for luxury marble or artistic geometric patterns, we’ve got you covered!

#Live URL
https://tiles-gallery-biia.vercel.app/

# Project Purpose
The main goal of this project is to provide a clean and user-friendly platform where users can discover various tile categories, search for their favorites, and view detailed information after a secure login.

# Key Features
Dynamic Tiles Gallery: Browse through a wide collection of tiles with a real-time search feature.
Secure Authentication: Easy login and registration using BetterAuth (Email/Password & Google Login).
Interactive Banner: A smooth, touch-friendly slider built with SwiperJS.
User Profile Management: Logged-in users can view and update their profile names and pictures.
Private Details: Tile details are protected—only community members can see the full specs!
Fully Responsive: Beautifully designed for mobile, tablet, and desktop screens.

# Tech Stack & Packages
This project is built using the latest web technologies:
*Framework: Next.js
*Styling: Tailwind CSS & DaisyUI
*Database: MongoDB
*Authentication: BetterAuth
*Slider: SwiperJS
*Icons: Lucide React
*Notifications: React Hot Toast
*Animations:* React Fast Marquee

# How to Run Locally
1. Clone this repository.
2. Install dependencies: `npm install`
3. Set up your `.env.local` file with MongoDB and BetterAuth credentials.
4. Run the development server: `npm run dev`
5. Open `http://localhost:3000` in your browser.
